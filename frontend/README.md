# Fortiq Frontend

Tactical Luxury design system frontend for Fortiq quantum-safe migration platform.

## Development

```bash
npm install
npm run dev
```

## Design System

- **Typography**: Bebas Neue (display), Cormorant Garamond (serif), Syne (UI), JetBrains Mono (data)
- **Colors**: Void black backgrounds, warm cream text, acid green accent
- **Layout**: 56px spine navigation, 48px top bar with live ticker
- **Patterns**: Section indexers, tactical data rows, editorial panel headers

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- Zustand (state)
- React Router
- Axios
- Recharts, react-force-graph-2d, CountUp
